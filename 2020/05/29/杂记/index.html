iOS 13.5 越狱！！！！！

我使用的是第一代 iPhone SE，A9 和 4 英寸巨屏已容不得我干大事了（比如说打王者什么的，再说我也不玩游戏）；128 GB 看似宽敞，但实际上放点电影就没地儿了。之前刷过 iOS 13，但嫌弃它还没有 iOS 12 越狱来的爽，就又刷回 12.4 搞越狱了。现在突然发现 13.5 的越狱都来了，而固件验证还没关，于是抓住最后的机会，让这部老机发光发热。

备份插件

这里用了 CyDown 自带的备份功能，备份文件的路径是/var/mobile/Library/Preferences/com.julioverne.cydown.backups.plist。

理论上来说，删除越狱状态并不会影响这里的文件，但保险起见，还是在电脑上复制一份。

删除越狱状态

在 unc0ver 的设置页面打开 Restore RootFS 开关，再回到主界面，会发现原来显示 Jailbreak 的按钮变成了 Restore RootFS，按下即可回到入狱状态，且保留原有数据。

下载 iOS 13.5 固件

学校里的千兆用惯了，家里的 200 兆显得极度缓慢。怎么会这样呢，我以前明明是用 ADSL 都能爽的人啊。

刷入固件

刷固件之前我就在想，之前嫌烦，刚装好系统就把 iTunes 直接 rm -rf 掉了，Apple Mobile Device 框架也是老版本，不知道还能不能刷机。果然，选好固件后就提示框架版本过低。

于是在苹果官网找到了 iTunes 12.8.2。但是，这个版本好像不是最新的，因为它居然提示最高只支持 macOS 10.13？？

于是我又在网上找到了能让 Mojave 强行安装 iTunes 12.8 的 Apple Script，想着装完再更新，代码如下。

display dialog "Please select iTunes 12.8 dmg file" buttons {"Ok"}
set theDMG to choose file with prompt "Please select iTunes 12.8 dmg file:" of type {"dmg"}
do shell script "hdiutil mount " & quoted form of POSIX path of theDMG
do shell script "pkgutil --expand /Volumes/iTunes/Install\\ iTunes.pkg ~/tmp"
do shell script "sed -i '' 's/18A1/14F2511/g' ~/tmp/Distribution"
do shell script "sed -i '' 's/gt/lt/g' ~/tmp/Distribution"
do shell script "pkgutil --flatten ~/tmp ~/Desktop/iTunes.pkg"
do shell script "hdiutil unmount /Volumes/iTunes/"
do shell script "rm -rf ~/tmp"

set question to display dialog "Install iTunes?" buttons {"Yes", "No"} default button 1
set answer to button returned of question
if answer is equal to "Yes" then
    do shell script "open ~/Desktop/iTunes.pkg"
    return
end if
if answer is equal to "No" then
    display dialog "Modified iTunes.pkg saved on desktop" buttons {"Ok"}
    return
end if

运行脚本后会在桌面生成 patch 后的安装包。

下面是照例两周总结。

上网乃人之常情

自从学校开始用钉钉上网认证，教室就不能每天都快乐了，唯一的机会是老师每天登微信传 ppt 的时候放下尊严跪求老师不要注销（虽然一般越这样说越不可能不注销）。

这时候就需要祭出技术手段了。众所周知，众所周知的软件不仅能做众所周知的事情，还能进行流量截断。钉钉上网认证的登录和注销页面的地址是不一样的，而注销页面地址是 192.168.180.100，那么就很棒了。

最后，把众所周知的软件加入开机自启，就离完美更进一步了。

班主任的暴怒

如上所述，教室时常没有网，于是我帮一群热爱死神 vs 火影的同学把 4399 上的 swf 下下来，又趁技术课下了个 flash player——新的快乐出现了！早上能早起了！中午能不睡觉了！晚自习之后不需要洗澡了！

这引起了班主任的注意。终于在上个星期，他忍不了了，扬言要给电脑设密码。一开始在座众人不以为意，当然这其中也包括我。

当天晚自习下课后，团支书从办公室回来跟我说，“老师叫你设个密码。”

居然是我来设？女少口阝可。（顺手备份了一下原 SAM 文件，哪天他突然变卦我又来不及破就尴尬了）

空调

*此处指寝室的空调

5 月了啊！闷热无比啊！受限于自身条件而无法怒发冲冠的宿管大爷可能不热，但这不代表同学们不热啊！6 个十几岁的热血少年蜗居于几十平米内的小房间，怎么可能像宿管住单人间一样不热啊！

于是有手有脚有脑子有裤兜有反侦察意识的同学（我）在发现寝室和教室的空调都是海尔，而且空调并没有断电，缺的只是遥控器后，在晚自习之后拿上了教室空调的遥控器，造福室友们。

然后就可以坐等宿管找上门来清凉一夏了。

合唱比赛

本质：校领导の迷惑行为

可能是学校想要让学生放松一下，但发现疫情期间把那么多人弄到礼堂去不太好，就搞了个录视频版的合唱比赛，要求就是录个视频，允许剪辑。

这好办啊，班里有微电影社的剪辑大佬，还有上学期我剪音频（指打开 Au 然后无限右键、平滑）在旁边边看边学的团支书，虽然是类信息技术活，但我只要动动嘴唱几句就完事了。

对吧？对吧？对吧？

前面确实是没我事的，选在校门口拍完视频，然后在班里录音，然后微电影社的剪辑大佬也就开始了。

但是，到团支书剪音频这里了，她居然叫我中午不回寝室待在教室跟她一起剪？？？我的技术水平有多菜她也不是没见过哇。不过，我技术水平菜不菜不重要，重要的是她的个性实在过于要强，她想要自己做主剪辑师，遇到不会的地方问我，总之她一个人操作。

听起来确实还八错。但我在教室里待了 10 分钟多，才发现这实属浪费人力资源，理由如下。

①我不是工具人

②我中午要睡觉

③有百度了还要我干啥

但当我意识到以上三点时，寝室已经关门了。得，坐教室里又闲又无聊又睡不着，不如去看哥们儿剪他们班的视频。**团支书居然叫人来叫我去给她继续做工具人？？

当然是继续和哥们儿聊天聊到下节课上课。一回教室才想起来是体育课。BUTT，团支书暴躁了，直接鼠标一扔，“你剪！”

彳亍口巴，我剪就我剪。

这好办啊，班里有微电影社的剪辑大佬，还有上学期我剪音频（指打开 Au 然后无限右键、平滑）在旁边边看边学的团支书，虽然是类信息技术活，但我只要动动嘴唱几句就完事了。

对吧？对吧？对吧？

最后总算是折腾完了，还趁着体育课做了众所周知的事情。

晚自习。剪辑大佬去找男朋友聊天了，“你对下音轨，顺便导出掉。”

彳亍口巴，我弄就我弄。
